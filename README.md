# yazi-plugins

A collection of plugins for [Yazi](https://github.com/sxyazi/yazi).

## Plugins

| Plugin | Description |
|--------|-------------|
| [trash-manager](trash-manager.yazi/) | Interactive trash manager with filtering, bulk operations, and thumbnail previews |
| [copy-content](copy-content.yazi/) | Copy the hovered file's contents to the system clipboard (text/empty only) |
| [archive-manager](archive-manager.yazi/) | Archive toolkit: preview listings, extract, and compress (7z/zip/tar variants) |
| [mounter](mounter.yazi/) | Interactive device mounter: USB drives, phones, SFTP/SMB via native yazi UI |

## Installation

```sh
ya pkg add Argus-Khan/yazi-plugins:plugin-name
```

See individual plugin READMEs for usage and keybindings.
