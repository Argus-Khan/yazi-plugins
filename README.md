# yazi-plugins

A collection of plugins for [Yazi](https://github.com/sxyazi/yazi).

## Plugins

| Plugin | Description |
|--------|-------------|
| [trash-manager](trash-manager.yazi/) | Interactive trash manager with filtering, bulk operations, and thumbnail previews |

## Installation

```sh
ya pkg add Argus-Khan/yazi-plugins:plugin-name
```

See individual plugin READMEs for usage and keybindings.
